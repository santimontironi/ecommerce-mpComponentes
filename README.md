### MpComponentes - E-commerce desarrollado con stack MERN para la venta de componentes de hardware (en desarrollo)

## Stack Tecnológico
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- Base de datos: MongoDB
- Pasarela de pago: MercadoPago API

## Características
- Catálogo de productos con búsqueda y filtros
- Carrito de compras persistente
- Sistema de autenticación de administrador
- Procesamiento de pagos con MercadoPago
- Panel de administración para gestión de productos
